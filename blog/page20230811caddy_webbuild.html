Caddy建站

使用Caddy快速搭建HTTPS的个人网站。不用手动申请证书，直接就是https网站。

  • 官网：Caddy官网
  • GitHub：caddyserver/caddy
  • GitHub：caddy官方文档

部署流程：

根据官方文档安装就可以，很简单。详见这里：快速安装
debian或ubuntu参考以下代码：
sudo apt install -y debian-keyring debian-archive-keyring apt-transport-https
curl -1sLf 'https://dl.cloudsmith.io/public/caddy/stable/gpg.key' | sudo gpg --dearmor -o /usr/share/keyrings/caddy-stable-archive-keyring.gpg
curl -1sLf 'https://dl.cloudsmith.io/public/caddy/stable/debian.deb.txt' | sudo tee /etc/apt/sources.list.d/caddy-stable.list
sudo apt update
sudo apt install caddy
当然也可以试一下docker部署，官方也有教程。

配置

最关键的当然是安装后的配置
Candy的默认配置文件是
/etc/caddy/Caddyfile
可以直接记事本打开修改。
下边是配置示例，实际使用请删除#和后边的注释：
hotine.wang { #要配置的域名
encode gzip #启用gzip压缩
log /opt/www/log/caddy.log #日志保存的位置（非必要）
root * /opt/www #网页文件所在的路径，根据实际情况修改
tls xxxxx@163.com #通知邮件地址（非必须）
file_server
}

movie.hotine.wang { #对这个域名进行重定向
redir https://hotine.wang/movie #网页重定向（任意url均可）
}

pan.hotine.wang,
cloud.hotine.wang { #这两个都使用同样的配置
redir https://hotine.wang/pan
}
修改完毕，重启下caddy即可。
在浏览器输入上述设置完毕的网址，查看是否生效。
